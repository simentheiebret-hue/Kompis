# Kompis Assets

Denne mappen skal inneholde appens bilder og ikoner.

For å få appen til å fungere, trenger du følgende filer:
- icon.png (1024x1024px)
- splash.png (1242x2436px)
- adaptive-icon.png (1024x1024px)
- favicon.png (48x48px)

Du kan opprette disse senere eller bruke Expo's standardikoner midlertidig.
